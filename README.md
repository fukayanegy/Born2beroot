## windows11環境の注意すべきコマンド
### reboot
通常のLinux環境ではrebootコマンドを使用して再起動を行うが、windows環境ではrebootコマンドは使用できない  
代わりに(sudo) system reboot を使用する  
### service
これは基本的にsystemコマンドで代用できる
